# AI-Based Anomaly Detection in 6G Networks
AI-based anomaly detection system for network traffic using Isolation Forest and UNSW-NB15 dataset.

This project implements an unsupervised machine learning model to detect anomalies in network traffic using the UNSW-NB15 cybersecurity dataset.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Isolation Forest

## Features
- Data preprocessing and feature engineering
- Isolation Forest anomaly detection
- Model evaluation using ROC-AUC and confusion matrix
- PCA visualization of anomalies

## Dataset
UNSW-NB15 dataset used for network intrusion detection.

## Results
The model identifies abnormal network patterns such as high connectivity attempts and suspicious traffic behavior.
